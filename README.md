# Movie App 🎬

A lightweight, browser-based application for managing a movie collection. This project was built to demonstrate proficiency in **Vanilla JavaScript DOM manipulation** and **CRUD** (Create, Read, Update, Delete) operations without the need for a backend database.

## 📌 Project Motivation
This project was developed as part of a web development class and serves as a core piece of my professional portfolio. The goal was to master how to dynamically handle user data and update the UI in real-time using only client-side technologies.

## ✨ Key Features
* **Dynamic Table Generation:** Submit a movie via the form, and it instantly appears in the table using JavaScript DOM methods.
* **Full CRUD Functionality:** * **Create:** Add new movies with ease.
    * **Read:** View all entries in a structured, clean table.
    * **Update:** Edit existing movie details directly in the list.
    * **Delete:** Remove entries from the collection with a single click.
* **Live Filtering:** Search and narrow down your list by **Title** or **Genre** using a responsive filter bar.
* **No Database Required:** All logic is handled in the browser session, making it fast and easy to deploy.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for forms and data display.
* **CSS3:** Custom styling for a clean, modern user interface.
* **JavaScript (ES6+):** Logic for event listeners, filtering, and DOM manipulation.

## 🚀 Getting Started

### Prerequisites
To run this project, you simply need a web browser. While not strictly required for the code to run, having **Node.js** installed allows you to use local development servers for a better experience.
